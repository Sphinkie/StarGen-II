Detailed Description

This is a generic class for creating a file with the solar system description.

This class must be instancied into derived classes. For instance derived class SG_File_TXT to produce a Text file, SG_File_XML to produce an XML file, and based on these examples, others derived classes can be created, and used in SG_SolarSystem.
