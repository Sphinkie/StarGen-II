◆ getBreathability()

int SG_Atmosphere::getBreathability ( )

This function determines if the atmosphere can be breathed or not.

This function uses figures on the maximum inspired partial pressures of Oxygen, other atmospheric and traces gases. From pages 15,16,18 of Dole's "Habitable Planets for Man".

Returns
The value returned is one of the value defined in the .h file (NONE, BREATHABLE, UNBREATHABLE, STINKING, POISONOUS)
